css selectors "::-moz-page-sequence"

The ::-moz-page-sequence CSS pseudo-element is a Mozilla extension that represents the background of a print preview.

Syntax

::-moz-page-sequence {
  /* ... */
}

Specifications

Not part of any standard.

See also

  • ::-moz-scrolled-page-sequence
